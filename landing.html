🚀 Welcome to DarkTraceX

An AI-powered tool to detect malicious (phishing) URLs using a trained Machine Learning model. Upload a CSV file of URLs and instantly find out which are safe and which are phishing.

✨ Key Features

  • 📂 CSV Upload – Upload a file of URLs for bulk phishing detection.
  • ⚡ Instant Predictions – Real-time classification as Legitimate or Phishing.
  • 📊 ML-Powered – Trained with advanced algorithms like Random Forest, Gradient Boosting, and Logistic Regression.
  • 📝 Download Results – Get predictions in an interactive HTML table or downloadable CSV.
  • 🔎 Explainable Features – Based on URL length, IP presence, SSL state, domain age, traffic, and more.
